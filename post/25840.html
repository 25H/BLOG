Arae Package

Arae 是一种简单的文件归档格式，它的工作方式类似于 Asar ，可以将所有文件连接在一起而不进行压缩，同时支持随机访问。


特征

  • 支持随机访问
  • 使用 JSON 存储文件信息
  • 支持文件加密
  • 支持大文件 (支持tb级大文件)
  • 高速解析 (结构简单的tb级文件小于100ms)
  • 支持文件合并
  • 支持文件分片
  • 使用简单
